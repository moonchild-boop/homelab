# Aisha Duze | Cybersecurity Portfolio

Welcome to my cybersecurity portfolio.

This repository documents hands-on projects completed as part of my journey into offensive security, security operations, governance, risk and compliance (GRC), identity and access management, and AI security.

The projects simulate real-world security assessments and investigations using an isolated homelab environment.

My focus is not simply exploiting vulnerabilities, but understanding security from end to end, demonstrating an understanding of how systems are attacked, detected, governed and improved.
